60 /* To use this file with libraries the following macro is useful:
61 
62  #define _ESTREAM_EXT_SYM_PREFIX _foo_
63 
64  This prefixes all external symbols with "_foo_".
65 
66  For the implementation of the code (estream-printf.c) the following
67  macros may be used to tune the implementation for certain systems:
68 
69  #define _ESTREAM_PRINTF_REALLOC foo_realloc
70 
71  Make estream_asprintf and estream_vasprintf use foo_realloc
72  instead of the standard realloc to allocate memory returned to
73  the caller. Note that foo_realloc needs to be C-90 compliant:
74  foo_realloc (NULL,n) is the same as a call to malloc(n) and
75  foo_realloc (a, 0) is the same as a call to free (a).
76 
77  #define _ESTREAM_PRINTF_EXTRA_INCLUDE "foo.h"
78 
79  This includes the file "foo.h" which may provide prototypes for
80  the custom memory allocation functions.
81  */

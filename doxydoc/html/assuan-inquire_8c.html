assuan_inquire: : An assuan context : The keyword used for the inquire : Returns an allocated buffer : Returns the length of this buffer : If not 0, the size limit of the inquired data.

A server may use this to send an inquire. r_buffer, r_length and maxlen may all be NULL/0 to indicate that no real data is expected. The returned buffer is guaranteed to have an extra 0-byte after the length. Thus it can be used as a string if embedded 0 bytes are not an issue.

Return value: 0 on success or an ASSUAN error code

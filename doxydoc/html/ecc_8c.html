nist_generate_key - Standard version of the ECC key generation. : A struct to receive the secret key. : Parameters of the curve. : Elliptic curve computation context. : Flags controlling aspects of the creation. : Only for testing : On success this receives an allocated MPI with the affine x-coordinate of the poblic key. On error NULL is stored. : Ditto for the y-coordinate.

Return: An error code.

The bits used by this function are PUBKEY_FLAG_TRANSIENT to use a faster RNG, and PUBKEY_FLAG_NO_KEYTEST to skip the assertion that the key works as expected.

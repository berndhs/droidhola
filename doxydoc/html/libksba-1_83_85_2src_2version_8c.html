§ ksba_check_version()

const char* ksba_check_version ( const char *  req_version)

ksba_check_version: : A string with a version

Check that the the version of the library is at minimum the requested one and return the version string; return NULL if the condition is not met. If a NULL is passed to this function, no check is done and the version string is simply returned. It is a pretty good idea to run this function as soon as possible, because it also intializes some subsystems. In a multithreaded environment if should be called before the first thread is created.

Return value: The version string or NULL

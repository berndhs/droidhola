_gcry_ecc_eddsa_genkey - EdDSA version of the key generation.

: A struct to receive the secret key. : Parameters of the curve. : Elliptic curve computation context. : Flags controlling aspects of the creation.

Return: An error code.

The only bit used by this function is PUBKEY_FLAG_TRANSIENT to use a faster RNG.
